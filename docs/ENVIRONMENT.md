## Backend (.env) — Managed on VPS

| Variable | Required | Description | Example |
|---|---|---|---|
| `DATABASE_URL` | ✅ | PostgreSQL connection | `postgres://tileserp:pass@localhost:5440/tileserp` |
| `JWT_SECRET` | ✅ | Access token signing (64+ chars) | Random string |
| `JWT_REFRESH_SECRET` | ✅ | Refresh token signing (64+ chars) | Random string |
| `JWT_EXPIRES_IN` | No | Access token TTL | `15m` |
| `JWT_REFRESH_EXPIRES_IN` | No | Refresh token TTL | `7d` |
| `PORT` | No | Backend port | `3003` |
| `NODE_ENV` | No | Environment | `production` |
| `CORS_ORIGIN` | No | Allowed origins | `https://tserp.digiwebdex.com` |
| `SMTP_HOST` | No | Email server | `smtp.gmail.com` |
| `SMTP_PORT` | No | Email port | `587` |
| `SMTP_USER` | No | Email username | `user@gmail.com` |
| `SMTP_PASS` | No | Email password | Gmail App Password |
| `SMTP_FROM` | No | Sender address | `user@gmail.com` |
| `BULKSMSBD_API_KEY` | No | SMS API key | API key from BulkSMSBD |
| `BULKSMSBD_API_URL` | No | SMS endpoint | `http://bulksmsbd.net/api/smsapi` |
| `BULKSMSBD_SENDER_ID` | No | SMS sender ID | Registered sender ID |
